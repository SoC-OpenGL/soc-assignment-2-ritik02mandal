Every folder has a makefile.
Run the "make" command to compile the code.
This will produce a single executable file.
To run it type in the terminal : ./"FILENAME"

Controls :

W : Moves Forward
S : Moves Backward
A : Moves Left
D : Moves Right
E : Moves Upward
Q : Moves Downward

Use Mouse to Look around
Use the Mouse scroll to zoom in and out

**For Q3 only**
I : Toggle Spotlight
O : Toggle Direction Light
P : Toggle Point Lights
B : Switch between Phong and Blinn-Phong shading models
N : Toggle to see the very fine NormalMap feature
